Built for people whose agent skills already live in too many folders, too many formats, and too many tools.

Features

A quieter control surface for messy local skill libraries.

Index

Normalize multiple skill layouts into one searchable local catalog.

Skills Manager scans known tool directories, parses frontmatter or headings, resolves real paths, and presents a unified index without hiding where the files really live.

Edit

Write back to the original file instead of exporting shadow copies.

What you edit is what your agents actually use. No sync ceremony, no secondary format.

Search

Retrieve skills by name, description, or raw content in one pass.

The index stays local, fast, and practical for real working directories.

Notes

Operational details worth knowing before you install.

macOS Gatekeeper

The current macOS package is not notarized yet, so the first launch may need a right-click Open flow or a quarantine flag removal.

Source of truth

The app does not invent a second content system. It indexes your files and writes changes back to the originals.

Scope

This release is focused on local scanning, search, and editing. Remote sync and registries can come later.
